FAN FAIR

In this project that two of us worked on, we developed a VR game using Unity XR Interaction Toolkit featuring a high-speed train, a ferris wheel, and toy vending machines. We collaborated closely at every stage of the project. I was responsible for designing environmental layouts, structures, machine algorithms, and animations.

DENTAL SIMULATION PROJECT

This project is being developed as part of an agreement between Gazi University's Faculty of Dentistry and the Institute of Informatics. The aim of this project is to enable dental students to learn and practice the correct usage of a dental X-ray machine. In this ongoing project, I am working as a developer focusing on aspects such as the operation of the machine, taking X-rays, and developing task systems.

BOWLING GAME

This project was developed using Unity 3D VR. I single-handedly developed this time-based bowling game for the purpose of showcasing and promoting VR usage at Sivas University during university promotional events. The game is designed to be a single-player experience focused on bowling.

HORROR GAME

We worked on this project as a team. Our team proudly developed an immersive FPS horror game set in a zombie apocalypse. My responsibilities included handling algorithms, assets, environmental layouts, and designs.
