101 EXTREME GAME

The 3D Polymorphic Study Buddy Game is planned as an interactive study aid to help users organize and manage their study sessions effectively. The project aims to feature a 3D polymorphic study buddy character that changes its appearance and behavior based on user progress and study habits. Additionally, YouTube API integration is intended to provide relevant educational videos to enhance the learning experience. Developed using Unity (C#) and Firebase, the game is designed to include study session planning, reminders, and gamification elements, making studying more engaging and interactive.
